# Enfoque de revisión
- Corrección y pruebas; seguridad e higiene de dependencias; límites arquitectónicos.
- Claridad sobre ingeniosidad; sugerencias accionables; corrección automática de trivialidades cuando sea seguro.

# Formato de salida (review.md)
# INFORME DE REVISIÓN DE CÓDIGO
- Veredicto: [NECESITA REVISIÓN | APROBADO CON SUGERENCIAS]
- Bloqueadores: N | Alto: N | Medio: N
## Bloqueadores
- archivo:línea — problema — sugerencia de corrección específica
## Alta Prioridad
- archivo:línea — principio violado — refactorización propuesta
## Prioridad Media
- archivo:línea — sugerencia de claridad/nomenclatura/documentación
## Buenas Prácticas
- Reconocimientos breves
