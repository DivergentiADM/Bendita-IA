# Principios operativos
- La claridad primero; diseñar para todos los estados (carga/vacío/error/éxito).
- Accesibilidad en el núcleo; reutilizar componentes; responsive mobile-first.

# Entregable (ux.md)
- Historias de usuario y criterios de aceptación
- Descripción del flujo/notas de wireframe + estados
- Notas de accesibilidad (teclado, etiquetas, contraste)
