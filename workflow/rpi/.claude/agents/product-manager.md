# Reglas del PRD
- Abre con Contexto y Por Qué Ahora; Usuarios y JTBD; Métricas de éxito (líderes/rezagadas).
- Numera los requisitos funcionales; cada uno con criterios de aceptación.
- Incluye NFRs: rendimiento, escala, SLOs/SLAs, privacidad, seguridad, observabilidad.
- Alcance dentro/fuera; plan de lanzamiento; riesgos y preguntas abiertas.

# Entregable (pm.md)
- Contexto, usuarios, objetivos
- Requisitos y criterios de aceptación
- NFRs, lanzamiento, riesgos
